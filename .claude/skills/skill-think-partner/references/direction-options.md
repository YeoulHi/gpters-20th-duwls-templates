# 방향성 선택지 가이드 (Direction Options)

Think Partner Skill의 Step 3에서 사용자가 선택하는 5가지 방향성에 대한 상세 설명입니다.

---

## 🎯 5가지 방향성 선택지

### (1) 강점 확대 (Deepen Expertise)

**설명:**
지금 잘하는 일을 더 깊이 있게, 더 높은 수준으로 발전시키는 방향

**핵심 질문:**
- "지금 하는 일 중 가장 잘하는 게 뭐야?"
- "이걸 더 깊이 있게 하면 어떻게 될 것 같아?"
- "시장에서 나의 강점이 가장 경쟁력 있나?"

**구체적 예시:**
```
현재: 영상 편집 자동화로 월 10시간 절감
↓ 강점 확대
미래: 영상 편집의 고급 기법 (AI 합성, 자동 자막 등) 습득
      → 더 높은 가격대의 클라이언트 확보
      → 월 수입 증대

또 다른 예:
현재: SNS 콘텐츠 분류 자동화
↓ 강점 확대
미래: 자동화 + 분석 기능까지 확장
      → 클라이언트에게 더 큰 가치 제공
```

**Step 4+ 심화 질문들:**

```
Q1: "지금 당신이 가장 강하다고 생각하는 분야가 뭔가요?"
→ 깊이 있는 분석으로 전문성 강화

Q2: "그 분야에서 당신만의 차별점이 뭘까요?"
→ 고유한 강점 발굴

Q3: "이 강점을 더 발전시키려면 어떤 기술이나 지식이 필요할까요?"
→ 학습 로드맵 수립

Q4: "강점이 깊어지면, 그것으로 뭘 할 수 있을까요?"
→ 실행 방안 모색

Q5: "이 방향을 위해 당신이 할 수 있는 첫 단계는?"
→ 구체적 실행 계획
```

**적합한 사람:**
- ✅ 이미 자신의 분야에서 어느 정도 성과를 내고 있는 사람
- ✅ 넓게 하기보다는 깊게 가고 싶은 사람
- ✅ "마이스터" 스타일의 전문가가 되고 싶은 사람

### (2) 영역 확장 (Expand Domain)

**설명:**
새로운 시장, 새로운 고객층, 새로운 분야로 비즈니스를 확대하는 방향

**핵심 질문:**
- "현재 서비스할 수 있는 다른 분야가 있을까?"
- "새로운 고객층은 어떤 문제를 가지고 있을까?"
- "내 기술을 다른 산업에 적용할 수 있을까?"

**구체적 예시:**

```
현재: 유튜브 채널의 SNS 관리 자동화
↓ 영역 확장
미래: 유튜브 → 팟캐스트, 블로그도 자동화
      → 더 많은 크리에이터 고객 확보
      → 월 수입 증대

또 다른 예:
현재: 프리랜서 본인을 위한 자동화
↓ 영역 확장
미래: 다른 프리랜서들도 자동화 필요
      → 자동화 솔루션 판매 사업
      → B2B 전환

또 다른 예:
현재: 한국 시장에서의 영상 편집
↓ 영역 확장
미래: 해외 클라이언트 확보
      → 국제 시장 진출
```

**Step 4+ 심화 질문들:**

```
Q1: "당신의 기술/경험을 필요로 하는 다른 분야가 있을까요?"
→ 새로운 시장 가능성 탐색

Q2: "새로운 영역에는 어떤 고객들이 있을까요?"
→ 타겟 고객층 정의

Q3: "그들의 가장 큰 pain point는 뭘까요?"
→ 니즈 파악

Q4: "당신이 해결해줄 수 있는 게 뭘까요?"
→ 가치 제안

Q5: "첫 고객 확보를 위해 할 수 있는 게?"
→ 초기 실행 계획
```

**적합한 사람:**
- ✅ 새로운 도전을 좋아하는 사람
- ✅ "다양한" 경험을 쌓고 싶은 사람
- ✅ 사업을 확장하고 싶은 사람

### (3) 역량 강화 (Build Capabilities)

**설명:**
기술, 지식, 자격증 등을 투자하여 새로운 역량을 갖추는 방향

**핵심 질문:**
- "부족하다고 느끼는 기술이 뭐야?"
- "배우고 싶은 새로운 분야가 있나?"
- "자격증이나 인증이 필요한가?"

**구체적 예시:**

```
현재: 영상 편집은 하는데, 기획/마케팅은 못함
↓ 역량 강화
미래: 마케팅 교육 수강 → 기획까지 할 수 있는 크리에이터
      → 더 높은 가치 제공
      → 자신의 채널도 성장

또 다른 예:
현재: 자동화는 하는데, 코딩은 못함
↓ 역량 강화
미래: Python 학습 → 더 복잡한 자동화 가능
      → 더 많은 클라이언트 수용
      → 기술 신뢰도 상승

또 다른 예:
현재: 프리랜서인데, 사업 운영에 자신 없음
↓ 역량 강화
미래: 회계/세무/법률 교육 → 체계적 사업 운영
      → 안정성 확보
```

**Step 4+ 심화 질문들:**

```
Q1: "앞으로 가장 필요한 역량이 뭐라고 생각하세요?"
→ 우선순위 파악

Q2: "그 역량을 어디서/어떻게 배울 수 있을까요?"
→ 학습 경로 탐색

Q3: "배우는 데 얼마나 걸릴 것 같은가?"
→ 실현 가능성 검토

Q4: "배운 후에는 뭘 할 수 있을까요?"
→ 미래 활용 상상

Q5: "첫 학습 단계로 할 수 있는 게?"
→ 첫 발걸음
```

**적합한 사람:**
- ✅ "배우는 것"을 좋아하는 사람
- ✅ 장기적 성장에 투자하고 싶은 사람
- ✅ 현재 부족함을 느끼는 사람

### (4) 사업 구조 변경 (Transform Business Structure)

**설명:**
프리랜서 혼자 하는 형태에서 팀이나 회사 형태로 전환하는 방향

**핵심 질문:**
- "혼자하기엔 일이 너무 많은가?"
- "팀을 꾸려서 더 큰 프로젝트를 하고 싶나?"
- "회사 형태로 체계화하고 싶나?"

**구체적 예시:**

```
현재: 자신이 영상 편집, SNS 관리 모두 처리
↓ 사업 구조 변경
미래: 팀 구성 (자신=크리에이티브, 팀원=기술/편집)
      → 더 많은 프로젝트 동시 진행
      → 월 수입 대폭 증대

또 다른 예:
현재: 프리랜서로 1인 사업 중
↓ 사업 구조 변경
미래: 회사 설립 → 여러 팀원 고용
      → 다양한 서비스 제공
      → 스케일업 가능

또 다른 예:
현재: 여러 프리랜서가 따로따로 일함
↓ 사업 구조 변경
미래: 협력사 네트워크 구성 → 공동 프로젝트
      → 시너지 발생
      → 각자 수입 증대
```

**Step 4+ 심화 질문들:**

```
Q1: "팀을 꾸린다면, 누가 필요할까요?"
→ 팀 구성 상상

Q2: "그들과 어떻게 일하고 싶으세요?"
→ 협업 방식 정의

Q3: "사업 구조 변경 시 가장 큰 과제는?"
→ 리스크 파악

Q4: "그 과제를 어떻게 극복할 수 있을까요?"
→ 해결 방안 모색

Q5: "첫 팀원이나 파트너는 누가 될 수 있을까요?"
→ 구체적 실행
```

**적합한 사람:**
- ✅ 이미 충분한 일감이 있는 사람
- ✅ 팀을 이끌고 싶은 사람
- ✅ 시스템화/스케일업을 꿈꾸는 사람
- ⚠️ 리더십과 관리 능력에 자신 있는 사람

### (5) 기타 (Others)

**설명:**
위의 4가지 선택지에 해당하지 않는 자신만의 독특한 방향

**핵심 질문:**
- "당신이 정말 하고 싶은 게 뭐야?"
- "현재 방향 4가지와 다른 뭔가가 있나?"
- "혹은 이들의 조합인가?"

**구체적 예시:**

```
예 1: 자동화 + 교육 결합
현재: 자동화를 직접함
새로운 방향: 다른 사람들을 가르치는 강사 되기

예 2: 프리랜서 + 개발 블로그 병행
현재: 프리랜서로만 일함
새로운 방향: 동시에 개발 콘텐츠로 영향력 만들기

예 3: 사업 + 사회활동
현재: 돈 버는 것에만 집중
새로운 방향: 무료로 자동화 팁을 공유하는 커뮤니티 운영

예 4: 완전 다른 분야로 전환
현재: 프리랜서 일
새로운 방향: 새로운 관심사 중심의 완전 다른 커리어
```

**Step 4+ 심화 질문들:**

```
Q1: "당신의 정말 궁극의 목표는 뭘까요?"
→ 본질적 욕망 파악

Q2: "그 목표를 달성하려면 뭐가 필요한가?"
→ 필요 조건 정의

Q3: "현재와 목표 사이의 간격은?"
→ 현실 파악

Q4: "그 간격을 줄이기 위한 첫 단계는?"
→ 실행 계획

Q5: "이 방향이 정말 맞는 것 같으세요?"
→ 최종 확인
```

**적합한 사람:**
- ✅ 창의적인 생각을 하는 사람
- ✅ 기존 틀을 벗어나고 싶은 사람
- ✅ 자신만의 길을 개척하고 싶은 사람

## 🔄 방향성 선택 프로세스

### Step 3: 사용자가 선택

```
당신이 말씀하신 [내용]을 보니,
다음 중 어느 방향에 관심이 있으신 것 같은데요.
맞는 것을 선택해주세요.

(1) 강점 확대
(2) 영역 확장
(3) 역량 강화
(4) 사업 구조 변경
(5) 기타
```

### Step 4+: 심화 질문 시작

선택한 방향에 맞는 Q&A 진행

### Step 6: 문서화

```
## 🧭 선택한 방향

**(1) 강점 확대**

이유: [사용자의 설명]

[심화 질문들과 답변]
```
