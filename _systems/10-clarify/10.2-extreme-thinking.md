# 10.2. 극단화 - 문제 본질 발굴

## 프로세스

### 1단계: 현재 상황
```
빈도: 주 ___회
소요시간: 회당 ___분
월 시간낭비: ___시간
```

### 2단계: 극단화 시나리오
```
만약 이 일이 "매일 50번" 반복된다면?
→ 월 ___시간 소요 (계산)
→ 이 상황에서 가장 문제는?

만약 팀 규모가 "5배"가 된다면?
→ 업무 연쇄 반응은?
→ 새로 생기는 문제는?
```

### 3단계: 본질 발굴
```
극단 상황에서의 비효율:
- _______________
- _______________
- _______________

자동화해야 할 구체적 항목:
1. _______________
2. _______________
```
