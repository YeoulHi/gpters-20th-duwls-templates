# 10.4. 에이전트선택가이드 - 경로 결정 (1분)

## 🎯 핵심 질문 하나

**"이 반복 업무의 문제점이 명확한가?"**

---

## 결정표

### ✅ YES → 경로 1 (20분)
명확한 문제 + 정량화 가능
```
→ 10.1 불편감지 (체크리스트)
→ 10.3 Task명확화 (5가지 질문)
→ 자동화 설계
```

### ⚠️ 모호함 → 경로 2 (30분, 추천)
문제가 명확하지 않음 + 더 깊게 이해 필요
```
→ 10.1 불편감지 (체크리스트)
→ 10.2 극단화 (10배 시나리오)
→ 10.3 Task명확화 (5가지 질문)
→ 자동화 설계
```

### ❓ 모름 → 경로 2 추천
시작하는 것이 낫다면 경로 2부터. 더 정확한 요구사항 발굴 가능.
