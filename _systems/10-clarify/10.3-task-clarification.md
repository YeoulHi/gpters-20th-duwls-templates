# 10.3. Task명확화 - 5가지 질문

## Q1: 작업의 이름은?
한 문장으로 설명 가능해야 함
```
예: "일일 리포트 생성"
예: "미팅 녹음 → 요약 변환"
```

---

## Q2: 현재 어떻게 하고 있나?
단계별 수동 과정 (5~7단계)
```
예: "Gmail 열기 → 메시지 읽기 → 우선순위 정렬 → 엑셀 입력 → CSV 저장 → 메일 발송"

현재 소요시간: ___분
```

---

## Q3: 얼마나 자주 반복하나?
정확한 빈도 (매일/주 N회/월 N회)
```
현재: 주 ___회 × ___분 = 월 ___시간
```

---

## Q4: 제약사항이 있나?
기술/환경/도구 제약
```
예: "한글 파일명 필수", "Windows만", "API 권한 제한"

제약사항:
- _______________
- _______________
```

---

## Q5: 성공 기준은?
측정 가능한 지표
```
예: "5분 → 30초로 단축"
예: "수동 개입 0%, 에러율 0%"

현재 ___분 → 자동화 후 ___분
```

---

## 📋 출력: 명확화된 Task

```
작업명: ___________________
현황: 월 ___시간 낭비, 주 ___회 반복
수동 과정: _______________
제약사항: _______________
성공 기준: 월 ___시간 절약 (ROI ___%)
```
